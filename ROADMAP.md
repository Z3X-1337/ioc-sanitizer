# Roadmap

## v0.2.0
- Add STIX 2.1 export.
- Add IDN and Punycode normalization.
- Add batch and directory input.
- Add configurable extraction policies.

## v0.3.0
- Add optional enrichment adapter interfaces.
- Add confidence and provenance metadata.
- Add analyst allowlists and suppression rules.
- Add streaming input for larger sanitized datasets.

## Non-goals
- Automatically declaring indicators malicious.
- Sending data to external enrichment services by default.
- Replacing analyst validation.
