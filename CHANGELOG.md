# Changelog

All notable changes to this project will be documented in this file.

The project follows Semantic Versioning.

## [0.1.0] - 2026-07-12

### Added
- Standard and commonly defanged IOC extraction.
- IPv4 and IPv6 validation.
- URL, domain, email, MD5, SHA-1, and SHA-256 support.
- Normalization and duplicate suppression.
- Flat JSON, grouped JSON, and CSV output.
- Source-form traceability.
- Unit and CLI tests across Python 3.10, 3.11, and 3.12.
- Installable command-line entry point.
